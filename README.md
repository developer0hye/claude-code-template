# claude-code-template

A template repository for Claude Code projects with `CLAUDE.md`, `METHODOLOGY.md`, `AGENTS.md`, and a reusable PRD authoring skill for Claude and OpenAI-compatible agents.

- Claude entry points:
  - `.claude/agents/prd-authoring.md`
  - `.claude/commands/prd.md`
- OpenAI-compatible agent entry point:
  - `.agents/skills/prd-authoring/`
